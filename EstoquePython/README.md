# 🛒 Sistema de Estoque em Python

Projeto desenvolvido com o objetivo de praticar **lógica de programação** e realizar a transição de **Portugol para Python**.

---

## 📌 Sobre o Projeto

Este sistema simula um controle básico de estoque de um supermercado, permitindo o cadastro e gerenciamento de produtos diretamente pelo terminal.

O foco principal do projeto foi consolidar conceitos fundamentais de programação, como:

* Estruturas de decisão (`if`, `elif`, `else`)
* Estruturas de repetição (`while`, `for`)
* Manipulação de listas
* Entrada e saída de dados
* Lógica de busca e comparação

---

## ⚙️ Funcionalidades

O sistema possui as seguintes funcionalidades:

* ✅ Cadastro de produtos (nome, preço e quantidade)
* 📋 Listagem de todos os produtos cadastrados
* 🔍 Busca de produto pelo nome
* 💰 Identificação do produto mais caro
* 📊 Cálculo do valor total do estoque
* 🚪 Encerramento do sistema

---

## 🧠 Lógica Aplicada

Durante o desenvolvimento, foram aplicados conceitos importantes como:

* Uso de listas paralelas para armazenar dados
* Percurso de listas com índices (`for` com `range`)
* Busca de elementos em listas
* Cálculo acumulativo (soma total do estoque)
* Comparação para encontrar maior valor
